[JSP Standard Tag Libarary(JSTL)]
- JSP 확장 태그이다.
- JSTL API와 구현체를 다운로드 받아야 한다.
- http://jstl.java.net 

[JSTL 사용법]
1) 사용할 태그 라이브러리를 가져온다.
<%@ taglib uri="라이브러리 URL" prefix="별명"%>

<별명:확장태그명 .....></별명:확장태그명>


[JSTL 라이브러리]
Core(http://java.sun.com/jsp/jstl/core, c) 라이브러리 
  => 변수 사용, 조건문, 반복문, URL 다루기, 예외처리, 출력
XML(http://java.sun.com/jsp/jstl/xml, x) 라이브러리
I18N(http://java.sun.com/jsp/jstl/fmt, fmt) 라이브러리
Database(http://java.sun.com/jsp/jstl/sql, sql) 라이브러리
Functions(http://java.sun.com/jsp/jstl/functions, fn) 라이브러리
