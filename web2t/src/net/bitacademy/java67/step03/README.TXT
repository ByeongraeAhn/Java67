- 퇴사처리
- 부서이동
- 부서해체

* Servlet 기술에서 웹 애플리케이션의 구성요소
1. 서블릿 컨테이너가 직접 호출하는 객체
1) Servlet
  - 클라이언트 요청 처리
  - 하나의 단위 업무를 처리
2) Filter
  - 요청을 처리하는 앞/뒤로 작업을 수행
  - 서블릿을 실행하기 전이나 실행한 후에 수행한다.
3) Listener
  - 특정 사건에 대해 작업을 수행한다.
  - 예) 웹 애플리케이션 시작/종료, 요청이 들어왔다, 세션이 생성되었다, ...
