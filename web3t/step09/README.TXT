[실습 목표: 파라미터 전달 자동화하기]
- 프론트 컨트롤러에서 페이지 컨트롤러의 메서드를 호출할 때 
  메서드의 파라미터를 분석하여 그 형식에 맞춰서 파라미터 값을 넣는다.

0. 실행하기 전에 자바 컴파일러 옵션에서 파라미터 변수 이름을 저장하도록 설정한다.
- 메서드의 파라미터 이름을 가지고 request.getParameter()를 호출할 것이기 때문에,
  .class 파일에 메서드의 파라미터 이름도 저장해야 한다.
- 그러나, 컴파일 기본 설정은 메서드의 파라미터 이름을 저장하기 않는다.
  단지 데이터 타입만 저장한다.
- 이번 실습을 위해 다음과 같이 파라미터 이름을 .class 파일에 저장하도록 설정하라!
- Preferences > Java > Compiler 화면에서
  [v] Store information about method parameters 항목을 체크하라!
